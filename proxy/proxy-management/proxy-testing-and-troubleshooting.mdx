This guide helps you verify that your proxy works correctly and troubleshoot common connection issues.

### How to test a proxy

1. Assign the proxy to your browser profile
2. Click the **refresh icon** next to the proxy location
3. Wait for the system to check the connection

If the proxy works correctly, the system will display:

- proxy IP address
- country or location
- connection status

If the proxy is working correctly, the status indicator will appear **green**.
