---
title: "Profile Export & Import"
---

This guide explains how to export and import profiles in Gologin and when to use each method.

Gologin supports profile import and export via **CSV files**.

## How to export profiles

1. **Select** one or more profiles from the All profiles page
2. Click **Export profiles**
3. **Save** the CSV file

<Frame>
  ![Profile Export](/images/profile_export.gif)
</Frame>

## How to import profiles

Profiles can be imported from the workspace menu.

1. **Click on the arrow** next to the Add profile button in the left menu
2. Click **Import profiles**
3. Upload a CSV file or paste CSV data
4. Click **Import**

<Frame>
  ![Profile Import](/images/profile_import-2.gif)
</Frame>

<Warning>
  If imported profiles are present in your All profiles list, they will just get updated. \
  Duplicates will not be created.
</Warning>
