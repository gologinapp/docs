Bulk actions in Gologin allow you to apply the same action to multiple profiles at once.

To use bulk actions, first select several profiles in the list using the checkboxes. After that, the bulk action panel appears at the top of the page.

### What you can do with bulk actions

**Run profiles**\
Select the profiles and click **Run profile** to launch all selected profiles.

**Run profiles with sync**\
Select the profiles and click **Run with Sync** to launch them with synchronization enabled.

**Stop profiles**\
Select the running profiles and click **Stop profile** to close them.

**Add profiles to a folder**\
Select the profiles, click **Add to Folder**, and choose the target folder.

**Assign or change proxies**\
Select the profiles, click **Proxy**, and add or update proxy settings for all selected profiles.

**Share profiles**\
Select the profiles, click **Share Profile**, and assign access to another user.

**Edit tags**\
Select the profiles, click **Edit tags**, and update tags for all selected profiles.

**Edit extensions**\
Select the profiles, click **Edit extensions** to manage extensions across the selected profiles.

**Edit bookmarks**\
Select the profiles and click **Edit bookmarks** to update bookmarks.

**Rename profiles**\
Select the profiles and click **Rename profiles** to rename them in bulk.

**Clone profiles**\
Select the profiles and click **Clone** to create copies of them.

**Transfer profiles**\
Select the profiles and click **Transfer** to move them to another workspace or user, if available in your setup.

**Export profiles**\
Select the profiles and click **Export profiles** to export them.

**Update fingerprint**\
Select the profiles and click **Update fingerprint** to refresh fingerprint settings.

**Delete profiles**\
Select the profiles and click **Delete** to remove them.

**Use automation**\
Select the profiles and click **Automation** to apply automation-related actions.

### Important

Bulk actions affect all selected profiles at once.\
Before applying any action, make sure the correct profiles are selected, especially when using **Delete**, **Transfer**, or **Update fingerprint**.
