## Hardware Requirements

- **RAM**: minimum 4GB 
- **Free disk space**: 1 GB 

## Supported Operating Systems

GoLogin is supported on the following operating systems:

### Windows
- Windows 11
- Windows 10
- Windows Server 2016

### macOS (including M1/M2 ARM chips)
- macOS 15 Sequoia (browser version 122 and above)
- macOS 14 Sonoma 
- macOS 13 Ventura 
- macOS 12 Monterey 
- macOS 11 Big Sur

### Linux
- Ubuntu
- Mint 

## Not Supported Systems

GoLogin does not support the following setups:

- Any 32-bit systems
- Windows 7, Windows 8, Windows 8.1, Windows Server 2012 (any revision) or lower
- macOS El Captain, Sierra, High-Sierra, Mojave, Catalina or lower
- iOS, iPhone and iPad included
- Chrome OS
