# Local Browser
Local browser in gologin means that you need to run browser on your machine and handle browser preparation and open with SDK that helps you to handle it


# Cloud Browser
Gologin provides cool feature that allow you to automate browser without running browser yourself!

# How does it work
<Steps>
  <Step title="Pass link right into your automation tool">
    You need to provide your dev API token
  </Step>
  <Step title="Gologin runs a browser on it's server and provides you websocket connection to the browser">
    You dont need to manage SDK, running browser yourself and monitor it
  </Step>
  <Step title="Use the connection with an automation tool">
    Use the browser as you want with your automation
  </Step>
  <Step title="Close browser">
    Or dont close it. We will close it ourself.
  </Step>
  <Step title="Enjoy your life">
    Gologin will manage your browser and anonimity. Automation tool will run the script on it while you do whatever you want!
  </Step>
</Steps>
