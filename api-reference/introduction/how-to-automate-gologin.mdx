# How does it work?

<Steps>
  <Step title="Provide Credentials">
    Give the SDK your developer token and the profile ID you want to run.
  </Step>
  <Step title="SDK Prepares Browser">
    The SDK takes care of downloading, preparing your profile, and starting the browser with the correct configuration.
  </Step>
  <Step title="Get Connection URL">
    SDK provides you with a websocket URL that automation tools can use to connect to the browser.
  </Step>
  <Step title="Connect Your Tool">
    Take the websocket URL and connect it to your automation tool of choice: Puppeteer, Selenium, Playwright, etc.
  </Step>
  <Step title="Control Through Code">
    The automation tool connects to the browser, allowing you to control it programmatically through your code.
  </Step>
</Steps>
