---
title: "Default profile settings"
---

**Default profile settings** let you define a template for new profiles. Anything you configure here is applied automatically when a new profile is created. Existing profiles are not affected — changes only apply going forward.

Not every setting is available in the default settings for now.

<Tip>
  These settings will only apply to all the profiles you're creating after the change, they won't be applied to already existing profiles.
</Tip>

How to get there:

1. Click your **workspace name** in the top-left corner.
2. Navigate to **Settings & Members**.
3. In the left sidebar, click **Default profile settings**.
